# OOP Practice Programs (Java)

## Description
This repository contains simple Java programs that demonstrate core Object-Oriented Programming (OOP) concepts.

## Concepts Implemented
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Technologies Used
- Java
- Object-Oriented Programming

## How to Run
1. Compile:
   javac FileName.java
2. Run:
   java FileName
